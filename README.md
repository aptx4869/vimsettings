这是我的 vimrc，备份及分享于 GitHub。请感觉自由去拷贝它。

## Usage

**Backup** your setting first, and then run
```
git clone git
cd vimsettings
git submodule init
git submodule update
```
I am useing a little bit strang Dvorak keyboard, so you should modify the keybindings to suit yourself.
